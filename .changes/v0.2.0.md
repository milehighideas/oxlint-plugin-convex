## v0.2.0 - 2026-06-29
### Added
* Read configuration from a dedicated `.convex-lint.json` — the single source of truth for the IDE, edit-time hook, commit gate, and CI. Ships a `.convex-lint.json.example`.
* Support a per-rule `errorRules` ratchet: only listed rules block on changed/edited files; all others stay warning-only so full-lint/CI never trip on pre-existing violations.
