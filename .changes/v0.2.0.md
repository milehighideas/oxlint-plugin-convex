## v0.2.0 - 2026-06-29
### Added
* Read configuration from a dedicated `.convex-lint.json` (single source of truth for the IDE, edit-time hook, commit gate, and CI), falling back to a `convexCheckConfig` block in `.pre-commit.json` for back-compat.
* Support a per-rule `errorRules` ratchet: only listed rules block on changed/edited files; all others stay warning-only so full-lint/CI never trip on pre-existing violations.
